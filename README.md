# Coursework-Full-Stack
A collection of projects I developed while working towards the Zenva Academy: [Full-Stack Web Developer Mini Degree](https://academy.zenva.com/product/full-stack-web-development-mini-degree/)

## MERN-Record-List
One of my ealiest React apps, a simple record list app that was created using MERN (MongoDB, Express, React, Node.js).

## GithubUserSearch
GitHub user search application built in Angular. Allows a user to login, input search terms, and receive results from the GitHub RESTful API.

## Reddit-Clone
A Reddit clone app based on one built for a Zenva React-Redux course

## Games-Website-in-Next.js
This is a games portfolio site built using Next.js

## Blog-Site-in-Vue.js
Here is a simple blogging app created using Vue.js. This is a work in progress. Future features will include either an SQL or MongoDB backend, as well as the ability to edit posts.

## User-Database-App
A simple application that allows you to login and add or delete users from a database. The app is built in Angular, and is populated by a Google Firebase db.

## Language-Learning-App
This is a language learning app built in Angular.

## Simple-Rest-API
A simple Rest API created using Node.js and Express. Includes user authentication functionality using Passport.js

## Language-Learning-Game-in-JS
This is a language learning game built using Javascript.
